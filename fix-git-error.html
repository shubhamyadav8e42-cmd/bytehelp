How to Fix "Not a Git Repository" Error

If you see the error message "fatal: not a git repository", it means Git is not initialized in your current folder.

Why This Error Happens

Git commands only work inside a Git repository. If you run git push or git status in a normal folder, Git will show this error.

Step-by-Step Fix

1. Check Your Current Folder

  pwd

2. Initialize Git (if not initialized)

  git init

3. Add Remote Repository

  git remote add origin https://github.com/USERNAME/REPO.git

4. Add and Commit Files

  git add .
  git commit -m "Initial commit"

5. Push to GitHub

  git push -u origin main

After following these steps, your error should be resolved.
